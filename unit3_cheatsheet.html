JavaScript Objects Cheat Sheet

What are objects in JavaScript?

JavaScript objects are a datatype which include a series of property/value pairs. Similar to associative arrays in other programming languages. Objects in JavaScript are indicated by curly brackets.

Properties and Values

Properties and values in a JavaScript object are separated by a ":". Each property/value pair is separated by a comma within the brackets. A JavaScript object is a collection of unordered properties. Properties can usually be changed, added, and deleted, but some are read only. Property is a variable attached to an object that is assigned a value. Property/value pairs are similar to key/value pairs for hashes in Ruby.

Example


  var firstObject = {
    property: "value",
    name : "first",
    number: 1
  };

Accessing object properties

There are several ways of accessing information within a JavaScript object. You can access either through bracket notation or dot notation. A significant difference, is that only bracket notation allows access to values using variables.

Example

  
  var object = { property : "value" };
  var x = 'property';
  
  Dot notation:
  object.property;
    ==> "value"
  object.x;
    ==> undefined
    
  Bracket notation:
  object["property"];
    ==> "value"
  object[x];
    ==> "value"

Modifying and adding object properties

To change the value of an object value you can also use either bracket or dot notation.

Example


  var emptyObject = {};
  
  Dot Notation:
  emptyObject.property = "value";
  emptyObject;
    ==> {property: "value"}
  emptyObject.property = "changed value";
    ==> {property: "changed value"}
  
  Bracket Notation:
  emptyObject["property"] = "value";
  emptyObject;
    ==> {property : "value"}
  emptyObject["property"] = "changed value";
  emptyObject;
    ==> {property : "changed value"}

Deleting object properties

To delete an object's property the "delete" keyword is used

Example


  var object = { property : "value" };
  object;
    ==> { property : "value" }
  delete object.property; 
  // or delete object["property"];
  object.property;
    ==> undefined
  object;
    ==> {}
